***Проект "Фрукты"***
----
В этом репозитории представлена простая программа на языке C, которая рисует различные фрукты.
____
__Программа из проекта__
----
Здесь представлены файлы из данного репозитория


**Фрукты**
``` C
#include <stdio.h>
#include <stdlib.h>
#include <locale.h>
int main() {
	setlocale(LC_CTYPE, "RUS");
	puts("    *   **                   ***  *****                ****   ****   ");
	puts("  *   *    *                *    *  *    *           *      *      * ");
	puts("    *   **                  ***    ****                **       **   ");
	puts("  *      *                *    *****    *            *    *   *    * ");
	puts("  *      *                *   *      *  *            *  -   *   -  * ");
	puts("*          *                ****    ***              * -   -   -   * ");
	puts("*          *                *    **   *              *    -  -  -  * ");
	puts("*          *                *    *    *              * -   -    -  * ");
	puts("  *      *                    **   **                  *    -  - *   ");
	puts("    ****                         **                      *******     ");
	puts("                                                                     ");
	puts("                                                                     ");
	puts("                  ***                           *                    ");
	puts("                *     **                      *   *                  ");
	puts("                *  *     *                  *   *                    ");
	puts("              *    ****                     *   *                    ");
	puts("              *   **  *                   *     *                    ");
	puts("              *   **  ***                 *       ***                ");
	puts("                *   ***   *               **           *******       ");
	puts("                  *   *   *                  **              *       ");
	puts("                  *   ***                       **         **        ");
	puts("                    **                              ******           ");
	return 100;
}
```

***Как запустить***
---
__1. Скопируйте репозиторий__
```
git clone [https://github.com/Hohopo00/Home.gid](https://github.com/Hohopo00/Home)
cd Home
```
__2. Скомпилируйте файл: Для компиляции фруктов:__
```
gcc fruits.c -o fruits
```
__3.Запустите файл__
```
./fruits
```
